# Start Your Data Scientist Career Path Here

This repository will take you step-by-step towards becoming a Data Scientist and proficient with the Python programming lanaguage. The repository is a progress through DataCamp's Career Track - Data Scientist with Python. The code and sequences present in this repository are part of exercises and assignments completed during the coursework at DataCamp. As I add chapters to the repository, I will also add links and supporting reading material to help prepare you for what is coming ahead. Let's get started!

#### Structure of the repository:

1. Code Files - Google Colab Notebooks
2. Readings - PDF in the Reading Sections
3. Additional Reading Material - Presented in the form of links to my Medium Articles for the supporting topics.
